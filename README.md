# Smart File Organizer (Python)

This project is a Python automation script that organizes files into categories such as PDFs, Images, Videos, and Documents.

## Features
- File organization by type
- Bulk renaming of files
- Large file separation
- Logging system

## Technologies Used
- Python
- os module
- shutil module

## How to Run
1. Install Python
2. Run the script:
   python main.py
